Features

All Block Types

Supports paragraphs, headings, lists, images, code, tables, toggles, callouts, embeds, and more.

Automatic Page Discovery

Any page shared with the integration becomes a published page on your site.

Serverless Architecture

Built on Netlify Functions. No servers to manage, scales automatically.

Smart Caching

Intelligent caching for fast load times while keeping content fresh.

Quick Setup

1. Clone and install

git clone [repo] && cd notion-cms && npm install

2. Configure environment

NOTION_TOKEN=secret_your_token
NOTION_BLOG_PAGE_ID=your_blog_page_id

3. Run locally

npm run dev
