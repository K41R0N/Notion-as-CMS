Quick Start

1. Clone this repository

git clone https://github.com/yourusername/notion-cms-starter.git
cd notion-cms-starter
npm install

2. Create Notion integration

Go to notion.so/my-integrations and create a new integration. Copy the token.

3. Configure environment variables

Copy .env.example to .env and add your credentials:

NOTION_TOKEN=secret_your_token_here
NOTION_DATABASE_ID=your_database_id
NOTION_BLOG_PAGE_ID=your_blog_page_id

4. Run locally

npm run dev

Visit localhost:8888

5. Deploy to Netlify

Push to GitHub and deploy with one click:

git push origin main

Then connect your repo in the Netlify dashboard.
